About this simulator

This simulator illustrates the core setup of the paper: how dynamical regimes at the micro level (inside each agent) relate to collective behavior at the macro level (across interacting agents).

Agents are placed in a 2D toroidal space and communicate only through binary light signals. Each agent contains an internal probabilistic reservoir; its light state is read from a designated neuron. At each macro step, agents receive egocentric exteroceptive observations of illuminated neighbors within vision radius $v_r$, and these inputs perturb their internal reservoir dynamics.

The key question is whether near-critical micro dynamics are sufficient to produce macro-level critical-like avalanche statistics, or whether collective behavior is primarily constrained by interaction topology and activity propagation.

Reservoir · micro level

Each agent contains an independent discrete-time stochastic reservoir with $N_{\mathrm{neurons}}$ binary neurons, following the branching-process formulation used in the paper. The neuron state is $s_{t,i}\in\{0,1\}$, and connectivity is a transmission-probability matrix $W_{i,j}\in[0,1]$. For each active pre-synaptic neuron, transmission events are sampled as $X_{t,i,j}\sim\mathrm{Bernoulli}(W_{i,j})$.

A neuron activates at $t+1$ if it receives at least one successful transmission:

$$s_{t+1,j}=\left[\sum_{i=1}^{N_{\mathrm{neurons}}} s_{t,i}X_{t,i,j}\ge1\right]$$

We remove self-connections ($W_{i,i}=0$) and use homogeneous transmission:

$$W_{i,j}=\begin{cases}p^{\mathrm{micro}}, & i\ne j\\0, & i=j\end{cases}$$

Under this assumption, the branching ratio $\sigma=(N_{\mathrm{neurons}}-1)p^{\mathrm{micro}}$ is the expected number of activations generated by one active neuron at the next micro-step. Intuitively: if $\sigma=1$, each active neuron replaces itself with about one new active neuron on average (activity is balanced); if $\sigma<1$, activity tends to shrink and die out; if $\sigma>1$, activity tends to expand and can become dense/saturated in finite systems. The equivalent critical point in terms of the transmission parameter is $p_c^{\mathrm{micro}}=\frac{1}{N_{\mathrm{neurons}}-1}$. Macro level · collective dynamics

At the macro level, the system is a population of $N_{\mathrm{agents}}$ light-signaling agents embedded in a 2D toroidal space. Agents interact only through local visual perception of illuminated neighbors, which induces a spatial interaction structure.

Each agent emits a binary light state $l_{t,i}\in\{0,1\}$. At each macro step, neighbors within vision radius $v_r$ are projected into $E$ egocentric angular sectors, producing a binary sensory vector. These $E$ inputs are clamped to designated reservoir input neurons, and a separate readout neuron determines the outgoing light state.

$$A_l(t)=\sum_{i=1}^{N_{\mathrm{agents}}} l_i(t),\quad \sigma_{\mathrm{eff}},\quad \mathcal{L}$$

Avalanche size and duration are extracted from contiguous active epochs of $A_l(t)$, then used to estimate $\sigma_{\mathrm{eff}}$ and the criticality score $\mathcal{L}$.

In the paper, sweeps over $(p^{\mathrm{micro}}, v_r)$ show that macro-level near-critical behavior depends on activity propagation through the interaction network, and is not determined by micro criticality alone.

Avalanche Extraction

Avalanche extraction follows the paper’s event definition on population activity:

$$A_l(t)=\sum_{i=1}^{N_{\mathrm{agents}}} l_i(t)$$

An avalanche starts at $t_s$ when $A_l(t_s-1)=0$ and $A_l(t_s)>0$, and ends at $t_e$ when $A_l(t_e)>0$ and $A_l(t_e+1)=0$. Its duration and size are:

$$T=t_e-t_s+1,\qquad S=\sum_{t=t_s}^{t_e}A_l(t)$$

Avalanches still active at the recording boundary are censored and excluded from analysis. In power-law fitting, the tail cutoff is $x_{\min}=2$ (for both $S$ and $T$); events below this cutoff are not used to estimate exponents.

Power-law fitting (paper vs live preview)

Paper methodology. In the paper, avalanche size and duration are fitted with a discrete power-law model $P(x)=x^{-\alpha}/\zeta(\alpha,x_{\min})$ for $x\ge x_{\min}$, following the Clauset framework. The cutoff $x_{\min}$ is selected by minimizing the Kolmogorov–Smirnov distance, then $\alpha$ is estimated by maximum likelihood on the fitted tail. The paper analysis also uses KS-based goodness-of-fit terms (and related fit diagnostics) in the criticality score $\mathcal{L}(p^{\mathrm{micro}},v_r)$.

Live viewer methodology. For real-time interaction, the live simulation uses a simplified estimator: fixed $x_{\min}=2$ and a fast closed-form discrete MLE for $\alpha$ from the currently accumulated avalanches. This preview fit does not run the full $x_{\min}$ search/KS optimization pipeline from the paper at every frame. The goal is responsiveness during exploration, not reproducing the full offline fitting workflow in real time.

Practical interpretation. Use the live fit as an immediate qualitative indicator while interacting with parameters. For quantitative claims, see the paper repository: https://github.com/nhbess/emergent-macro-criticality.
